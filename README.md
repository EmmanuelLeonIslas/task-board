# Task Board - Frontend

A modern, responsive task management application built with TypeScript, Vite, and Tailwind CSS.

## Live Demo

**[View Live Application](https://task-board-eight-olive.vercel.app/)**

## Features

- ✅ Create, read, update, and delete tasks
- 🔄 Organize tasks in three columns: Pending, In Progress, and Completed
- 🌓 Dark mode support
- 📱 Fully responsive design
- ✨ Smooth animations and transitions
- 🔒 Client-side validation with Zod
- 💾 Persistent data storage with MongoDB

## Tech Stack

- **Framework:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Validation:** Zod
- **Deployment:** Vercel
